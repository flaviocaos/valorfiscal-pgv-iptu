## 🚀 Como Usar

```bash
git clone https://github.com/seu-usuario/valorfiscal-pgv-iptu.git
cd valorfiscal-pgv-iptu
npm install
npm start
```

## ✨ Funcionalidades

- Dashboard com métricas, gráficos, mapa de calor, ranking e auditoria
- Parâmetros tributários configuráveis (alíquotas, CUB, fatores)
- Simulador individual com passo a passo detalhado
- Simulação em lote com upload/export CSV
- Detalhamento educativo do cálculo PGV
- Geração de 5 functions PostgreSQL prontas para uso
